## 重要数字

- `fuse_rrf` 36 µs/次 → 单核可处理 ~28K RRF 融合/秒（远超实际检索吞吐需求）
- `HybridPipeline` 28 µs/次 → 算子层零开销
- `ragas_faithfulness` 49 µs/次（mock judge）→ 真实 LLM judge 延迟由 vendor 决定，算子框架本身不构成瓶颈
